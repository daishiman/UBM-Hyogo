## 各未タスクの詳細

### UN-01: Cloudflare R2 ストレージの設定

**概要**: ファイルアップロード・画像配信のための Cloudflare R2 バケット設定

**前提条件**:
- `ubm-hyogo` アプリにファイルアップロード機能が実装される段階

**コスト**: 無料枠 10GB ストレージ・1,000万読み/月

---

### UN-02: Cloudflare KV セッションキャッシュの設定

**概要**: セッション管理・設定キャッシュのための KV Namespace 設定

**前提条件**:
- 認証機能実装後

**コスト**: 無料枠 100,000読み/日・1,000書き/日

---

### UN-03: Cloudflare Zero Trust の認証設定

**概要**: `/admin` 等の管理画面に対する SSO アクセス制御

**前提条件**:
- 管理画面が実装された後

---

### UN-04: WAF / Rate Limiting ルールの設定

**概要**: API エンドポイントへの DoS 対策・不正リクエストフィルタリング

**前提条件**:
- 本番トラフィックが発生した後

---

### UN-05: カスタムドメイン設定

**概要**: `ubm-hyogo-web.pages.dev` → 独自ドメインへの移行

**前提条件**:
- ドメイン取得後
- M-02（URL 最終確定）

---

## 対応方針

- UN-01〜UN-07 は別タスクとして `doc/` 配下に仕様書を作成する
- 優先度 高 の UN-05 は早期対応を推奨
- UN-03 / UN-04 はセキュリティ強化フェーズで対応
