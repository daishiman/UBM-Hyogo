# UBM兵庫支部会 GAS Prototype

Google Apps Script Webアプリ向けの仮プロトタイプです。

現行の実装ターゲットは `apps/web` + `apps/api` + D1 であり、このフォルダは画面感の確認用参照に留める。

## 含まれるもの

- `Code.gs`: GASのエントリポイント
- `index.html`: HTML/CSS/JavaScript一体型の単一ページUI

## できること

- 概要、メンバー一覧、メンバー詳細、閲覧専用プロフィール、管理メンバー、管理開催日、スキーマ設定の画面遷移
- タグ辞書と Gemini 候補確認の画面
- Google Form の 31 項目を反映したプロフィール詳細と本人編集フォーム
- モックデータでの一覧・詳細表示
- `localStorage` による編集内容の永続化
- モバイルでも見やすい 2 列ナビとセクションジャンプ
- 本人のみ編集できるプロフィール画面、閲覧専用の詳細画面、管理者は表示 / 参加 / 開催日のみ操作
- 管理者による表示 / 非表示の切り替え
- 管理者による開催日の追加
- 管理者による各メンバーの参加日の付与 / 解除
- 仮のプロフィール編集
- ログイン画面なし
- 閲覧は青系、本人編集は緑系で色分けし、役割が直感的に分かるUI

## GASでの使い方

1. Google Apps Scriptで新規プロジェクトを作成
2. `Code.gs` を貼り付ける
3. `index.html` をHTMLファイルとして追加する
4. `デプロイ > 新しいデプロイ > ウェブアプリ` で公開する

## 注意

- 認証、API、DB接続は未実装です
