## PASS Boundary

Local implementation may reach `PASS_BOUNDARY_SYNCED_RUNTIME_PENDING` when fixture export, tests, typecheck, lint, leakage scan, schema validation, and SSOT sync pass but production read-only export has not been approved.

State vocabulary is fixed:

| State | Meaning |
| --- | --- |
| `LOCAL_PASS` | fixture export / tests / typecheck / lint passed |
| `RUNTIME_PENDING` | production read-only export awaits explicit approval |
| `RUNTIME_PASS` | production export evidence captured after approval |

## Completion
